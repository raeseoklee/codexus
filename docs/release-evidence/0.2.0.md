No real-model E2E is required before tag publish for `0.2.0`. The release
promotes local, deterministic evidence surfaces and does not add live model
execution, automatic prompt injection, app health authority, or plugin
always-on supervision.

## Deferred Surfaces

The following remain outside the `0.2.0` stable contract:

- live app-instance health authority and automatic cleanup;
- live `cx autopilot run`;
- active relay engine spawning;
- Desktop app-server live attachment;
- automatic context or prompt injection;
- LSP protocol-server lifecycle and automatic project LSP application;
- plugin always-on supervision.
