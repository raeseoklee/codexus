## Manual E2E Sign-Off

No real-model E2E is required before tag publish for `0.2.3`. The release adds
cache-only advisory update notices, bounded evidence summaries, explicit wiki
approval listings, and documentation/demo artifacts. It does not add live model
execution, automatic installation mutation, automatic prompt injection, app
health authority, cleanup authority, or new completion authority.

## Deferred Surfaces

The following remain outside the stable contract:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, leases, and retry;
- LSP protocol-server lifecycle and automatic project LSP application;
- Desktop app-server live attachment;
- automatic context or prompt injection;
- app health authority and automatic cleanup.
