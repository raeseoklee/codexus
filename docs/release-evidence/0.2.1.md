## Manual E2E Sign-Off

No real-model E2E is required before tag publish for `0.2.1`. The release adds
local, deterministic readiness and adapter-reporting evidence surfaces. It does
not add live autopilot execution, automatic scheduler ownership, LSP
protocol-server lifecycle, live Desktop attachment, automatic prompt injection,
or new completion authority.

## Deferred Surfaces

The following remain outside the stable contract:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, leases, and retry;
- LSP protocol-server lifecycle and automatic project LSP application;
- Desktop app-server live attachment;
- automatic context or prompt injection;
- app health authority and automatic cleanup.
