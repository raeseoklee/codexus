## Automated Evidence

- Release prep:
  - Package version: `0.2.5`
  - Planned tag: `v0.2.5`
  - Publish path: tag-triggered GitHub Actions trusted publishing
- Release scope since `0.2.4`:
  - `cx app instance profile doctor --json` reports read-only app-instance
    descriptor preflight evidence without starting or controlling an app.
  - `cx app instance observe --json` reports split process, heartbeat, port,
    HTTP projection, and log signals without becoming health, cleanup, control,
    or completion authority.
  - `cx app instance evidence collect --json` records a bounded metrics, logs,
    and loopback HTTP evidence pack for one `instanceId`.
  - `cx evidence status --json` provides an experimental project evidence
    dashboard over app-instance evidence, deterministic wiki status, and LSP
    detection status.
  - `cx evidence check --gate --json` aggregates existing repo, wiki, LSP, and
    release gates without becoming semantic completion authority.
  - `cx evidence export --target <path> --json` writes an explicit markdown/JSON
    projection for human review, rejects symlinked workspace-local targets,
    reports file-byte hashes, and does not auto-commit.
  - Feature reference, implementation status, remaining-work backlog, roadmap
    kanban, project wiki current-state, changelog, and release evidence docs
    were refreshed before tag publish.

## Manual E2E Sign-Off

No real-model E2E is required before tag publish for `0.2.5`. The release adds
bounded evidence operations and app-instance observation helpers only. It does
not add live model execution, automatic installation mutation, automatic prompt
injection, app health authority, cleanup authority, or new completion authority.

## Deferred Surfaces

The following remain outside the stable contract:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, leases, and retry;
- LSP protocol-server lifecycle and automatic project LSP application;
- Desktop app-server live attachment;
- automatic context or prompt injection;
- app health authority and automatic cleanup.
