## Automated Evidence

- Release prep:
  - Package version: `0.2.6`
  - Planned tag: `v0.2.6`
  - Publish path: tag-triggered GitHub Actions trusted publishing
- Release scope since `0.2.5`:
  - `install.sh --help` and `install.sh -h` now print usage before any
    Node/npm probe, npm install, filesystem write, bin-link mutation, or Codex
    skill adapter installation.
  - Unknown installer arguments fail closed with exit code `2` instead of
    falling through to installation.
  - README install guidance now tells users they can inspect installer usage
    with `sh install.sh --help`.

## Manual E2E Sign-Off

No real-model E2E is required before tag publish for `0.2.6`. The release is an
installer safety hotfix and does not add live model execution, app health
authority, cleanup authority, prompt injection, or new completion authority.

## Deferred Surfaces

The following remain outside the stable contract:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, leases, and retry;
- LSP protocol-server lifecycle and automatic project LSP application;
- Desktop app-server live attachment;
- automatic context or prompt injection;
- app health authority and automatic cleanup.
