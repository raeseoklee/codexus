## 수동 E2E Sign-Off

`0.1.3` tag publish 전 새 real-model E2E는 필요하지 않습니다. 새 surface는
experimental이며 gate 뒤에 있습니다. App instance stop은 owner identity verification을
요구하고, automation live dispatch는 explicit approval을 요구하며, app instance
lifecycle projection은 cleanup, health, completion authority를 얻지 않습니다.
`0.1.0` manual E2E가 실제 Codex 실행 baseline으로 남아 있습니다.
