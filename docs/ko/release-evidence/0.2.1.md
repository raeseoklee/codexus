## Manual E2E Sign-Off

`0.2.1` tag publish 전 real-model E2E는 필요하지 않습니다. 이 release는 local,
deterministic readiness와 adapter-reporting evidence surface를 추가합니다. Live
autopilot execution, automatic scheduler ownership, LSP protocol-server lifecycle,
live Desktop attachment, automatic prompt injection, 새로운 completion authority는
추가하지 않습니다.

## Deferred Surfaces

다음 항목은 stable contract 밖에 남아 있습니다:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, lease, retry;
- LSP protocol-server lifecycle과 자동 project LSP 적용;
- Desktop app-server live attachment;
- automatic context 또는 prompt injection;
- app health authority와 automatic cleanup.
