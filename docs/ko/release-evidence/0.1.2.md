## 수동 E2E Sign-Off

`0.1.2` tag publish 전 새 real-model E2E는 필요하지 않습니다. 이 release는
experimental evidence surface, release integrity, 문서, session control-plane
projection을 중심으로 한 additive 0.1.x patch입니다. `0.1.0` manual E2E가 실제 Codex
실행 baseline으로 남아 있고, `0.1.2`는 installed-package smoke, local release gate,
trusted-publishing workflow evidence로 커버합니다.
