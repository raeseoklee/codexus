## 수동 E2E Sign-Off

`0.1.4` tag publish 전 새 real-model E2E는 필요하지 않습니다. 새 surface는
fake/Codexus-owned stdio process 위의 experimental proof harness입니다. Desktop stdio에
attach하지 않고, Desktop turn을 시작하지 않고, transcript 값을 저장하지 않으며,
completion authority를 얻지 않습니다.
