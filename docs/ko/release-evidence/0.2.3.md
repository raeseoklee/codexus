## Manual E2E Sign-Off

`0.2.3` tag publish 전 real-model E2E는 요구하지 않습니다. 이 release는 cache-only
advisory update notice, bounded evidence summary, 명시적 wiki approval listing,
documentation/demo artifact를 추가합니다. Live model execution, automatic
installation mutation, automatic prompt injection, app health authority, cleanup
authority, 새로운 completion authority는 추가하지 않습니다.

## Deferred Surfaces

다음 항목은 stable contract 밖에 남아 있습니다:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, lease, retry;
- LSP protocol-server lifecycle과 automatic project LSP application;
- Desktop app-server live attachment;
- automatic context 또는 prompt injection;
- app health authority와 automatic cleanup.
