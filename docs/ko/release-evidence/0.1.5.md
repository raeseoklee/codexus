# 0.1.5 Release Evidence

[English](../../release-evidence/0.1.5.md)

상태: `0.1.5` stable release candidate

이 파일은 `0.1.5` app-instance HTTP evidence probe patch를 위한 redacted release
evidence artifact입니다. Raw run ledger, transcript, stdout dump, token, private
project를 드러내는 local path, model prompt는 커밋하지 않습니다. Release checklist를
증명하는 데 필요한 bounded evidence만 기록합니다.

## 자동 증거

- Release prep:
  - Package version: `0.1.5`
  - Planned tag: `v0.1.5`
  - Publish path: tag-triggered GitHub Actions trusted publishing
- `0.1.4` 이후 release scope:
  - Experimental app-instance HTTP evidence probe:
    `cx app instance evidence probe --instance-id <id> [--url <loopback-url>]`.
  - Probe는 running Codexus-owned worktree app instance의 dev-server response
    evidence를 bounded, redacted artifact로 기록합니다.
  - Evidence는 `app-instance-observation` artifact에 연결되지만 control, health,
    cleanup, completion authority가 되지 않습니다.
  - 실행 중이 아닌 instance는 request 없이 `unavailable`로 기록됩니다.
  - Non-loopback URL과 30초 초과 timeout 값은 거부됩니다.
  - Stable JSON contract: frozen stable field 제거/재정의 없음.
- Tag publish 전 local release gate:
  - Verification id: `verification_20260603_045740_580400`
  - Result: `passed`
  - Commands: `npm run typecheck`, `node --test tests/app-instance.test.ts`,
    `npm test`, `node codex/skills/codexus/scripts/cx.mjs repo check --gate --json`,
    `npm run package:smoke`, `git diff --check`
  - Release-candidate gate:
    `node codex/skills/codexus/scripts/cx.mjs release check --version 0.1.5 --gate --json`

## 수동 E2E Sign-Off

`0.1.5` tag publish 전 새 real-model E2E는 필요하지 않습니다. 새 surface는 이미
Codexus-owned인 app instance 위의 experimental loopback HTTP evidence probe입니다.
Desktop turn을 시작하지 않고, browser를 열지 않고, app health를 주장하지 않으며,
completion authority를 얻지 않습니다.
