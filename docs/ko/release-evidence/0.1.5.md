## 수동 E2E Sign-Off

`0.1.5` tag publish 전 새 real-model E2E는 필요하지 않습니다. 새 surface는 이미
Codexus-owned인 app instance 위의 experimental loopback HTTP evidence probe입니다.
Desktop turn을 시작하지 않고, browser를 열지 않고, app health를 주장하지 않으며,
completion authority를 얻지 않습니다.
