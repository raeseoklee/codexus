# 0.2.8 Release Evidence

[English](../../release-evidence/0.2.8.md)

Status: `0.2.8` release prep in progress

이 파일은 stable release-channel simplification patch를 위한 redacted release-prep
evidence artifact입니다. Raw run ledger, transcript, stdout dump, token, private
project local path, model prompt를 커밋하지 마세요. Release checklist를 증명하는 데
필요한 bounded evidence만 기록합니다.

## Automated Evidence

- Release prep:
  - Package version: `0.2.8`
  - Planned tag: `v0.2.8`
  - Publish path: tag-triggered GitHub Actions trusted publishing
- `0.2.7` 이후 release scope:
  - Stable live release sign-off는 이제 npm `latest`, GitHub latest, GitHub
    Release `install.sh` asset만 gate합니다.
  - npm `next`는 명시적 prerelease 전용 channel로 취급합니다. Stable release
    completion은 더 이상 `next >= latest` 또는 maintainer
    `npm dist-tag add ... next` mutation을 요구하지 않습니다.
  - `releaseIntegrity.npm.nextDistTagAction`은 compatibility를 위해 live JSON output에
    남아 있으며 stable sign-off에서는 `not_applicable`을 보고합니다.
  - Local publish helper는 stable fallback publish에서 `latest`만 검증합니다.
    `publish:next`는 명시적 prerelease path로 남습니다.

## Manual E2E Sign-Off

`0.2.8` tag publish 전 real-model E2E는 요구하지 않습니다. 이 release는
release-channel policy와 release-integrity hardening slice입니다. Live model execution,
app health authority, cleanup authority, prompt injection, 새로운 completion authority를
추가하지 않습니다.

## Deferred Surfaces

다음 항목은 stable contract 밖에 남아 있습니다:

- live `cx autopilot run`;
- unattended cron/gateway scheduler ownership, lease, retry;
- LSP protocol-server lifecycle과 automatic project LSP application;
- Desktop app-server live attachment;
- automatic context 또는 prompt injection;
- app health authority와 automatic cleanup.
