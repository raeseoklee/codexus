# Code of Conduct

[English](../../../CODE_OF_CONDUCT.md)

## 기준

Codexus contributor는 discussion을 technical, specific, respectful하게 유지해야
합니다. Direct하고 critical한 기여는 좋지만, evidence, tradeoff, 실제 작업에
초점을 맞춰야 합니다.

기대하는 행동:

- 기술적 의견 차이를 concrete reasoning으로 설명합니다.
- feedback을 actionable하게 유지합니다.
- 서로 다른 경험 수준과 context를 존중합니다.
- personal attack, harassment, discriminatory language를 피합니다.

허용되지 않는 행동:

- insult, threat, sustained disruptive comment
- sexualized language 또는 imagery
- public/private channel에서의 harassment
- 허가 없는 private information 공개

## Enforcement

Maintainer는 이 code를 위반하는 comment, issue, pull request, contribution을 수정,
숨김, 삭제할 수 있습니다. 심각하거나 반복적인 위반은 project space에서 temporary
또는 permanent ban으로 이어질 수 있습니다.

Conduct concern은 GitHub를 통해 maintainer에게 보고하세요. Security-sensitive
report는 [SECURITY.md](../../../SECURITY.md)를 따릅니다.
